## 📡 API 端点速记

```
# 用户
GET  /api/admin/users      - 获取列表
POST /api/admin/users      - 创建
PUT  /api/admin/users/:id  - 编辑
DELETE /api/admin/users/:id - 删除

# 宠物
GET  /api/admin/pets       - 获取列表
POST /api/admin/pets       - 创建
PUT  /api/admin/pets/:id   - 编辑
DELETE /api/admin/pets/:id - 删除

# 商品
GET  /api/admin/products   - 获取列表
POST /api/admin/products   - 创建
PUT  /api/admin/products/:id - 编辑
DELETE /api/admin/products/:id - 删除

# 其他 (仅读和删除)
GET/DELETE /api/admin/matches
GET/DELETE /api/admin/conversations
GET/DELETE /api/admin/messages
GET/DELETE /api/admin/notifications
```

## 🔧 故障排查速记
